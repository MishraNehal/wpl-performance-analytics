## 📌 Project Overview
End-to-end data analysis and machine learning project on the **Women's Premier League (WPL)** cricket tournament covering all 4 seasons (2023–2026), with predictions for the **2027 season**.

## 🎯 Key Highlights
- 📊 Analyzed **65+ matches** across 4 seasons
- 🏆 Predicted **WPL 2027 Champion** using Gradient Boosting
- 🏏 Forecasted **Top Batter & Bowler** for 2027 using Linear Regression
- 🪙 Toss impact analysis — does the toss decide the match?
- 🔥 Head-to-head win matrix across all teams

## 📂 Dataset
- Source: [Kaggle — WPL India 2023-2025](https://www.kaggle.com/datasets/aravindas01/wpl-india)
- Source: [Kaggle — WPL 2026](https://www.kaggle.com/datasets/vishwassrivastava1/womens-premier-league-wpl-2026-datasets)
- Seasons: 2023, 2024, 2025, 2026

## 📈 Analysis Performed
1. Data cleaning & merging across 4 seasons
2. Team win/loss analysis
3. Season-wise performance trends
4. Win method analysis (runs vs wickets)
5. Toss impact analysis
6. Player of the match insights
7. Top batter & bowler analysis (WPL 2026)
8. Head-to-head win matrix

## 🤖 ML Models Used
| Model | Purpose | Accuracy |
|---|---|---|
| Random Forest | Match winner prediction | ~69% |
| Gradient Boosting | WPL 2027 champion forecast | ~75%+ |
| Linear Regression | Player performance projection | Trend-based |
